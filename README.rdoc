== 成年後見人業務サポートサービス

* 後見人が行う、高齢者のお世話内容、財産状況、健康情報などの記録業務がスマホで簡単に行え、関係者にリアルタイムで情報共有できるサービス

（主な機能）<br>
* 高齢者との面談の記録<br>
→面談直後に、高齢者のご家族にその内容がメールで共有される（開発中）

* 年間の記録が一覧表出力（開発中）<br>
→家庭裁判所への報告業務が簡単になる<br>

== 開発環境
* Rails 4.2.8
* Ruby 2.3.1p112
* gem 2.5.1
* PostgreSQL

== 用語
* guardian：後見人
* ward：被後見人
* record:訪問記録
